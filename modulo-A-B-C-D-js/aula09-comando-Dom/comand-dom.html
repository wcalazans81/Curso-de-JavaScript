Comando Dom em JavaScript

O comando Dom só roda quando o Index está rodando no navegador

A ávore Dom serve para localizar os elementos dentro do seu HTML.

Para acessar a ávore Dom uasr os seguintes comandos em JavaScript

window.document.write('Olá Mundo!')
window.document.write(window.document.charset) //mostrar na tela de acordo com a árvore dom
window.document.write(window.navigator.appName) //buscar configurações da minha página
window.document.write(window.document.URL) //buscar configurações da minha página navegdor e url

Para acessar um elemento pontual na árvore dom usa-se comandos específicos
Como por exemplo:

var p1 = window.document.getElementsByTagName('p')[1]//selecionar tags pelo nome e mostra texto do p
window.document.write('Elaine Linda ' + p1.innerText)
var p1 = window.document.getElementsByTagName('p')[1]//selecionar tags pelo nome e mostra texto do p
p1.style.color = 'red' //muda o estilo do parágrafo

Seleção por id do elemento

var s = window.document.getElementById('msg') //seleção do elemento por id
s.style.background = 'green'// muda cor da seleção
var s = window.document.getElementById('msg') //seleção do elemento por id
s.innerText = 'Elaine Linda'//muda texto do elemento via javascript

Seleção por nome do elemento

var n1 = window.document.getElementsByName('gms')[0] //seleção do elemento por nome
s.innerText = 'Elaine Linda'//muda texto do elemento via javascript
n1.style.background = 'green'// muda cor da seleção

Seleção por class do elemento

var c1 = window.document.getElementsByClassName('smg')[0] //seleção do elemento por class
c1.innerText = 'Elaine Linda'//muda texto do elemento via javascript
c1.style.background = 'green'// muda cor da seleção

Seleção por Seletor do elemento

var c1 = window.document.querySelector('section.msg') //seleção do elemento por seletor
c1.innerText = 'Elaine Linda'//muda texto do elemento via javascript
c1.style.background = 'green'// muda cor da seleção
